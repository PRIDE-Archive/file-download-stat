## Introduction

A pipeline to calculate number of file downloads in PRIDE archive

## Usage

To Create the Environment:

`conda env create -f environment.yml`

To Activate conda :

`conda activate file_download_stat`

## Debugging

Check Nextflow:

`nextflow -version`
